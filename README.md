# HR Analytics Dashboard

## Project Overview
This project analyzes employee attrition, workforce distribution, and employee performance trends using Power BI.

The dashboard helps understand key HR metrics and supports data-driven decision-making.

## Tools Used
- Power BI
- Excel

## Key Features
- Employee Attrition Analysis
- Department-wise Employee Distribution
- Gender Diversity Insights
- Salary Analysis
- Performance Trends
- Interactive Filters and Visualizations

## Key Insights
- Identified departments with high attrition
- Analyzed employee distribution across job roles
- Compared salary and performance metrics
- Improved workforce trend visualization
